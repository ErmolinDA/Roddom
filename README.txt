проект запускается в jupyter notebook, файл roddom

py -m pip install -r requirements.txt - установка библиотек для работы, запускать из командной строки из папки с проектом
prepared_data - файл с подготовленными данными(удалены неполные записи)
results - папка с таблицами результатов
geodata.json - файл с разметкой районов

Создание geodata.json
1. На сайте geojson.io сформировать районы
2. В разделе Table создать new column с именем id 
3. Присвоить каждому району уникальное имя(номер или текст)
4. Содержимое раздела JSON скопировать в geodata.json
